# XylarJava v0.0.0.2-ALPHA

XylarJava is a lightweight Minecraft launcher focused on offline accessibility and integrated modding support. It allows users to manage game instances and visual customization without requiring a constant internet connection.
Core Features

    Offline Mode: Full support for launching Minecraft without authentication server dependency.

    Modpack Integration: Built-in downloader and manager for modpacks, streamlining the installation of complex mods.

    Offline Skin System: Integrated mod-based skin rendering that displays user skins from local storage while offline.

    Resource Efficiency: Optimized Java core designed for fast startup times and minimal system impact.
